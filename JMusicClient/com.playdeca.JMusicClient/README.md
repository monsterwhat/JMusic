# JMusicClient: A Decentralized Music Streaming Application

JMusicClient is a modern, web-based music streaming application built with Java and the Quarkus framework. It allows you to manage and play your local music library through a clean and intuitive web interface.

**Key Features:**

*   **Web-Based UI:** Access your music library from any device with a web browser.
*   **Music Library Management:** Organize your songs into playlists.
*   **Real-time Updates:** Uses WebSockets for instant updates on playback state and other events.
*   **Decentralized Music Sharing:** Leverages a torrent-like system for sharing and discovering new music within a peer-to-peer network.
*   **REST API:** Provides a comprehensive API for controlling the application and managing your music.

**Efficiency Note:** This software is designed to be 50% more efficient than regular streaming services, offering a streamlined experience.

**Technology Stack:**

*   **Backend:** Java, Quarkus
*   **Frontend:** HTML, CSS, JavaScript
*   **Database:** H2
*   **Real-time Communication:** WebSockets
*   **Peer-to-Peer:** bt-core (a BitTorrent library)
